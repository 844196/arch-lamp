# arch-lamp

## Usage

```shell-session
$ git clone https://github.com/844196/arch-lamp && cd arch-lamp
$ vagrant up
$ vagrant ssh
```

## Provision

* PHP7
* composer
* Apache
* MariaDB
* phpMyAdmin
* tool
    * vim
    * zsh
    * git
    * tmux
* dotfiles(<https://github.com/844196/dotfiles>)
    * run [bootstrap](https://github.com/844196/dotfiles/blob/master/bootstrap)
